我们安装了git之后，都要先配置以下git工作环境。git提供了git config的工具，专门用来配置或读取相应的工作环境变量。

这些配置一般会存在三个地方：

  • /etc/gitconfig：全局配置（针对所有用户）
  • ~/.gitconfig：全局配置（针对某个用户）
  • .git/config：局部配置（针对某个目录/项目）
